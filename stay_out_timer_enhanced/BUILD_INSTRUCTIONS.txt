ИНСТРУКЦИЯ ПО СБОРКЕ ПРИЛОЖЕНИЯ НА WINDOWS

1. Установите Python 3.8 или новее с https://python.org

2. Скачайте архив с исходным кодом

3. Откройте командную строку и перейдите в папку с проектом

4. Установите зависимости:
   pip install -r requirements.txt

5. Установите Pillow для создания иконки (опционально):
   pip install Pillow

6. Запустите скрипт сборки:
   python build_exe.py

7. Готовый исполняемый файл будет создан в папке dist/
   и архив с программой будет доступен в корне папки

Примечание: Приложение будет собрано как Windows .exe файл без консоли
