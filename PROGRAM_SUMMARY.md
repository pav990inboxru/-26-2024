# Таблицы рынка Stay Out - Программа для создания таблиц покупок и продаж

## Описание проекта

Программа создана Harper_IDS для игрового сообщества IgromanDS. Это упрощенное приложение для создания таблиц покупок и продаж с доски объявлений в игре Stay Out. Приложение оформлено в стиле S.T.A.L.K.E.R. для удобства использования даже детьми.

## Файлы проекта

1. **stay_out_market_table_creator.py** - Основной файл программы с GUI-интерфейсом
2. **START_Таблицы_рынка_Stay_Out.py** - Файл запуска программы с понятным названием
3. **launch_program.py** - Альтернативный лаунчер для запуска программы
4. **requirements.txt** - Файл зависимостей (Pillow)
5. **README_RU.md** - Документация на русском языке

## Функциональность

- **Создание категорий товаров** (например, "Грибы")
- **Добавление товаров** с указанием:
  - Названия товара
  - Минимальной цены покупки на доске
  - Цены продажи (если сам продаешь)
  - Места продажи и кладовщика
  - Ссылки на товар
- **Возможность вставки изображений/иконок**
- **Вкладка библиотеки** для просмотра созданных таблиц
- **Вкладка со ссылками** на игру и вики:
  - https://so-wiki.ru/
  - https://store.steampowered.com/app/1180380/Stay_Out/
  - https://steamcommunity.com/id/Harper_IDS/
- **Автосохранение данных** при закрытии программы
- **Экспорт таблиц в CSV формат**

## Особенности интерфейса

- Стиль оформления в стиле S.T.A.L.K.E.R. с темной цветовой схемой
- Удобные вкладки для навигации
- Простой и интуитивно понятный интерфейс
- Все элементы управления логично расположены
- Поддержка русского языка

## Требования

- Python 3.x
- Библиотеки: tkinter, PIL (Pillow), json, os, webbrowser, csv

## Установка и запуск

1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите программу одним из способов:
   - `python START_Таблицы_рынка_Stay_Out.py`
   - `python launch_program.py`
   - `python stay_out_market_table_creator.py`
