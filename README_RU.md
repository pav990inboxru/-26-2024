# Таблицы рынка Stay Out

Программа для создания таблиц покупок и продаж с доски объявлений в игре Stay Out.
Оформление в стиле S.T.A.L.K.E.R. для удобства использования даже детьми.

## Функциональность

- Создание категорий товаров (например, "Грибы")
- Добавление товаров с указанием:
  - Названия товара
  - Минимальной цены покупки на доске
  - Цены продажи (если сам продаешь)
  - Места продажи и кладовщика
  - Ссылки на товар
- Возможность вставки изображений/иконок
- Вкладка библиотеки для просмотра созданных таблиц
- Вкладка со ссылками на игру и вики
- Автосохранение данных
- Экспорт таблиц в CSV формат

## Как использовать

1. Запустите программу с помощью файла `START_Таблицы_рынка_Stay_Out.py`
2. На вкладке "Создание таблиц":
   - Добавьте новую категорию (например, "Грибы")
   - Выберите категорию из списка
   - Заполните данные товара и нажмите "Добавить/Обновить товар"
3. На вкладке "Библиотека" можно сохранять и загружать таблицы
4. На вкладке "Ссылки" находятся полезные ресурсы по игре

## Требования

- Python 3.x
- Библиотеки: tkinter, PIL (Pillow), json, os, webbrowser, csv
