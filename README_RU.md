# Таймер для Stay Out

Программа создана разработчиком Harper_IDS для сообщества IgromanDS

## Описание
Это приложение для отслеживания игрового времени в игре Stay Out. Программа учитывает специфику игрового времени, где 1 реальная секунда равна ~6.87 игровых секунд.

## Функции
- Старт/Пауза/Стоп/Сброс таймера
- Редактирование времени
- Настройка скорости игрового времени
- Вкладка помощи с инструкциями
- Автоматическое сохранение последнего времени

## Установка и запуск

### Требования
- Python 3.6 или выше
- Библиотека tkinter (обычно входит в стандартную установку Python)

### Установка
1. Убедитесь, что у вас установлен Python 3
2. Tkinter обычно входит в стандартную библиотеку Python, но в некоторых системах может потребоваться установка:
   - Ubuntu/Debian: `sudo apt-get install python3-tk`
   - CentOS/RHEL: `sudo yum install tkinter` или `sudo dnf install python3-tkinter`
   - Windows: Tkinter входит в стандартную установку Python

### Запуск
1. Скачайте файл `stay_out_timer_full.py`
2. Откройте терминал или командную строку
3. Перейдите в папку с файлом
4. Выполните команду: `python stay_out_timer_full.py`

## Использование
1. На вкладке "Таймер" вы найдете основные элементы управления
2. Нажмите "Старт" для начала отсчета игрового времени
3. Используйте "Пауза", чтобы временно остановить таймер
4. "Стоп" останавливает таймер и сохраняет текущее время
5. "Сброс" обнуляет таймер
6. "Редактировать время" позволяет установить произвольное значение
7. На вкладке "Настройки" можно изменить скорость игрового времени
8. Вкладка "Помощь" содержит подробные инструкции

## Особенности
- Стандартная скорость игрового времени: 6870 мс (1 реальная секунда = ~6.87 игровых секунд)
- Программа автоматически сохраняет последнее время при закрытии
- При повторном запуске восстанавливает предыдущее состояние
