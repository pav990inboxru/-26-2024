# Таймер для игры Stay Out (Исправленная версия)

## Описание
Это приложение создано для отслеживания игрового времени в игре Stay Out с исправленной логикой расчета и улучшенным интерфейсом. Программа учитывает специфику игрового времени, которое отличается от реального времени.

**Программа создана разработчиком Harper_IDS для сообщества IgromanDS**

## Особенности
- Точное отслеживание игрового времени с учетом соотношения реального и игрового времени
- Исправленная логика расчета игрового времени (ранее время устанавливалось рандомно)
- Возможность паузы, остановки, сброса и редактирования времени
- Настройка скорости игрового времени
- Удобное окно для установки времени (отдельные поля для часов, минут, секунд)
- Цветные кнопки для лучшей различимости
- Возможность запуска по нажатию клавиши Enter
- Автосохранение игрового времени при закрытии программы
- Автоматическая загрузка сохраненного времени при запуске
- Отдельная вкладка с инструкциями для новичков
- Сохранение настроек между запусками

## Управление
- **Зеленая кнопка "Старт"** или клавиша **Enter** - Начать отсчет игрового времени
- **Оранжевая кнопка "Пауза"** - Приостановить таймер (время сохраняется)
- **Синяя кнопка "Стоп"** - Остановить таймер (время не сбрасывается)
- **Красная кнопка "Сброс"** - Сбросить таймер до 00:00:00
- **Фиолетовая кнопка "Изменить время"** - Открытие окна для установки времени

## Настройки
- Скорость игрового времени: регулировка продолжительности игровой секунды (по умолчанию 6870 мс)
- Настройки внешнего вида: изменение цветов интерфейса
- Возможность сброса к стандартным настройкам

## Инструкция для новичков
Вкладка "Помощь" содержит подробное руководство по использованию приложения для пользователей любого уровня.

## Технические детали
- Стандартная скорость игрового времени: 6870 мс (1 реальная секунда = примерно 6.87 игровых секунд)
- Приложение использует tkinter для графического интерфейса
- Все настройки сохраняются в файл settings.json
- Игровое время автоматически сохраняется в файл game_time.json при закрытии

## Запуск
Для запуска приложения просто запустите исполняемый файл `StayOutTimer` (на Linux) или `StayOutTimer.exe` (на Windows).

## Сборка исполняемого файла
См. файл BUILD_INSTRUCTIONS_FIXED.txt для инструкций по сборке на Windows.
