# История изменений Stay Out Timer

## v1.0.1 - Исправления и улучшения

### Исправленные ошибки:
- Исправлена логика расчета игрового времени при установке времени
- Исправлена ошибка, из-за которой время устанавливалось рандомно
- Исправлена логика работы таймера при паузе и возобновлении

### Новые функции:
- Удобный ввод времени через отдельное окно с полями ввода для часов, минут и секунд
- Возможность запуска таймера по нажатию клавиши Enter
- Цветные кнопки для лучшей различимости:
  - Зеленая: Старт
  - Оранжевая: Пауза
  - Синяя: Стоп
  - Красная: Сброс
  - Фиолетовая: Изменить время
- Автосохранение игрового времени при закрытии программы
- Автоматическая загрузка сохраненного игрового времени при запуске

### Технические улучшения:
- Улучшена структура переменных для более точного отслеживания игрового времени
- Добавлена функция автосохранения состояния таймера
- Добавлена обработка закрытия окна для корректного сохранения данных
