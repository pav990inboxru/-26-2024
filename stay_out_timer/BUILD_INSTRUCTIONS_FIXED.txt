ИНСТРУКЦИИ ПО СБОРКЕ ИСПОЛНЯЕМОГО ФАЙЛА Stay Out Timer (Исправленная версия)

1. УСТАНОВКА ЗАВИСИМОСТЕЙ:
   - Установите Python 3.7 или выше
   - Установите PyInstaller: pip install pyinstaller

2. ПРОВЕРКА КОРРЕКТНОСТИ КОДА:
   - Убедитесь, что main.py запускается без ошибок: python main.py

3. СБОРКА ИСПОЛНЯЕМОГО ФАЙЛА:
   - В командной строке перейдите в папку с проектом
   - Выполните команду: pyinstaller StayOutTimer.spec
   - Или используйте: pyinstaller --onefile --windowed --icon=icon.ico main.py

4. РЕЗУЛЬТАТ:
   - Исполняемый файл будет создан в папке dist/
   - Название файла: StayOutTimer.exe
   - Приложение будет запускаться без видимой консоли
   - Используется иконка в стиле STALKER

5. ВАЖНЫЕ ОСОБЕННОСТИ ИСПРАВЛЕННОЙ ВЕРСИИ:
   - Исправлена логика расчета игрового времени
   - Удобный ввод времени через отдельное окно
   - Цветные кнопки для лучшей различимости
   - Возможность запуска по нажатию Enter
   - Автосохранение игрового времени при закрытии
   - Автоматическая загрузка сохраненного времени при запуске

6. ТЕСТИРОВАНИЕ:
   - Запустите приложение
   - Проверьте работу всех функций
   - Убедитесь, что таймер корректно считает игровое время
   - Проверьте функцию установки времени
   - Проверьте автосохранение при закрытии и загрузку при запуске
