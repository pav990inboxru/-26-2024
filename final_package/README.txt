ТАЙМЕР ДЛЯ STAY OUT

Программа создана разработчиком Harper_IDS для сообщества IgromanDS

ОПИСАНИЕ:
Это полнофункциональное приложение для отслеживания игрового времени в игре Stay Out.
Программа учитывает специфику игрового времени, где 1 реальная секунда равна ~6.87 игровых секунд.

ОСНОВНЫЕ ФУНКЦИИ:
• Старт/Пауза/Стоп/Сброс таймера
• Редактирование времени
• Настройка скорости игрового времени
• Вкладка помощи с инструкциями
• Автоматическое сохранение последнего времени

ФАЙЛЫ В ПАКЕТЕ:
- stay_out_timer_full.py - основной файл программы
- INSTRUCTIONS_RU.txt - инструкция по установке и запуску
- stay_out_timer.spec - файл спецификации для PyInstaller
- build_exe.py - скрипт для создания .exe файла

ЗАПУСК ПРОГРАММЫ:
1. Убедитесь, что у вас установлен Python 3.6+
2. Запустите: python stay_out_timer_full.py

СОЗДАНИЕ .EXE ФАЙЛА:
1. Установите PyInstaller: pip install pyinstaller
2. Запустите: python build_exe.py
   или выполните: pyinstaller --onefile --windowed --name "StayOutTimer" stay_out_timer_full.py

ПРИМЕЧАНИЯ:
- Стандартная скорость игрового времени: 6870 мс
- Программа автоматически сохраняет и восстанавливает время между сеансами
- Интерфейс полностью на русском языке
- Три вкладки: Таймер, Настройки, Помощь
