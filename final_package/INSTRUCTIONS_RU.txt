ИНСТРУКЦИЯ ПО УСТАНОВКЕ И ЗАПУСКУ ПРОГРАММЫ

Таймер для Stay Out
Программа создана разработчиком Harper_IDS для сообщества IgromanDS

1. УСТАНОВКА PYTHON
-------------------
Для работы программы необходим Python 3.6 или выше.

Windows:
- Скачайте Python с официального сайта python.org
- При установке обязательно поставьте галочку "Add Python to PATH"
- Tkinter входит в стандартную библиотеку Python, дополнительная установка не требуется

Linux (Ubuntu/Debian):
- Откройте терминал и выполните: sudo apt-get install python3 python3-tk

Linux (CentOS/RHEL):
- Выполните: sudo yum install python3 tkinter
  или: sudo dnf install python3-tkinter

macOS:
- Python обычно установлен по умолчанию
- Если нет, установите через homebrew: brew install python3

2. ЗАПУСК ПРОГРАММЫ
-------------------
- Скачайте файл stay_out_timer_full.py
- Откройте терминал или командную строку
- Перейдите в папку с файлом: cd путь/к/папке
- Запустите программу: python stay_out_timer_full.py
  или: python3 stay_out_timer_full.py

3. СОЗДАНИЕ .EXE ФАЙЛА (ДЛЯ WINDOWS)
----------------------------------
Если вы хотите создать исполняемый .exe файл:

1. Установите PyInstaller:
   pip install pyinstaller

2. Перейдите в папку с программой:
   cd путь/к/папке

3. Создайте .exe файл:
   pyinstaller --onefile --windowed --name "StayOutTimer" stay_out_timer_full.py

4. Готовый файл будет в папке dist/StayOutTimer.exe

4. ФУНКЦИИ ПРОГРАММЫ
-------------------
- Старт: запускает отсчет игрового времени
- Пауза: временно останавливает таймер
- Стоп: останавливает таймер и сохраняет текущее время
- Сброс: обнуляет таймер
- Редактировать время: позволяет установить произвольное значение
- Настройки: изменение скорости игрового времени
- Помощь: подробные инструкции по использованию

5. ОСОБЕННОСТИ
--------------
- Стандартная скорость игрового времени: 6870 мс (1 реальная секунда = ~6.87 игровых секунд)
- Программа автоматически сохраняет последнее время при закрытии
- При повторном запуске восстанавливает предыдущее состояние
